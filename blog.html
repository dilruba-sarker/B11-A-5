    • - Question-2: What is the difference between innerHTML, innerText, and textContent ?

    1. innerHtml returns all text, including html tags, that is contained by an element.
    2. innerText returns all text contained by an element and all its child elements.
    3. textContents is all text contained by an element and all its children that are for formatting purposes only.

    -Question-4: What is event bubbling in the DOM?

    Event bubbling is a behavior in JavaScript where an event that is triggered on a child element will also be triggered on its parent elements all the way up to the document object.
